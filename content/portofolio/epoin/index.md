e-Poin atau N3TS Piket merupakan aplikasi piket untuk mencatat data pelanggaran, izin, dan dispensasi siswa/i untuk sekolah.

### Teknologi
- PHP 7
- MySQL
- Bootstrap CSS v4

Source code at [Github](https://github.com/n3tsconnect/epoin).
